# Quick Start Guide

## Run the Application in 3 Steps

### Step 1: Prerequisites
- Install Docker Desktop
- Make sure ports 80, 5000, and 27017 are free

### Step 2: Start Application
```bash
cd "d:\DevOps Final Exam"
docker-compose up -d
```

### Step 3: Access Application
Open browser: http://localhost:80

## Stop Application
```bash
docker-compose down
```

## View Logs
```bash
docker-compose logs -f
```
